== Description ==

Adds Bootstrap classes to Ninja Forms.

Note: This plugin does not load Bootstrap CSS or JavaScript itself. You should use a Bootstrap enabled theme or add Bootstrap to the theme yourself.

== Installation ==

1. Upload the `bootstrap-ninja-forms` folder to the `/wp-content/plugins/` directory
1. Activate the Bootstrap Ninja Forms plugin through the 'Plugins' menu in WordPress
